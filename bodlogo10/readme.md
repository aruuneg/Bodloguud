# Бодлого 10

## Бодлогын өгүүлбэр  
`1`-ээс `n` хүртэлх бүхэл тоонуудын пермутаци үүсгэх бөгөөд тухайн индекс `i`-тэй (1-ээс эхэлдэг) элемент `P[i]` нь дараах нөхцлийг хангасан байх ёстой:
Өөрөөр хэлбэл, бүрэн пермутаци үүсгэхдээ элемент бүрийн байрлалаас ялгаатай байдал нь `k` байх ёстой. Хэрэв ийм пермутаци байх боломжгүй бол `-1` гэж буцаана.

## Холбоос:  
[https://www.hackerrank.com/challenges/absolute-permutation](https://www.hackerrank.com/challenges/absolute-permutation)

## Шийдэл:  
Эхлээд `k == 0` байвал хариу нь 1-ээс `n` хүртэлх тоонуудын дараалал байна.  
Хэрэв `n % (2 * k) != 0` бол пермутаци үүсгэх боломжгүй тул `-1` гэж буцаана.  
Харин эсрэг тохиолдолд, блок бүрт `2 * k` элемент байх бөгөөд эхний `k` элементэд `+k` нэмэх, дараагийн `k` элементэд `-k` хасах байдлаар шийднэ. Энэ горимыг блок бүрээр ээлжлэн гүйцэтгэж, хүссэн пермутаци үүсгэнэ.
