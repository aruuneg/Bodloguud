## Бодлогын тайлбар

## Шийдэл:

1. Тэмдэгт мөрийн баруун талын бага зэрэг буурсан индексийг олох.  
2. Тэр индексээс баруун талд байх том тэмдэгтийг олох.  
3. Тэр хоёр тэмдэгтийг солих.  
4. Солигдсон тэмдэгтээс хойшхи хэсгийг урвуу эрэмбэлэх.
